# GitHub Setup Instructions

Your repository is ready to push to GitHub!

## Option 1: Create New Repository on GitHub

1. Go to https://github.com/new
2. Repository name: `ai-code-review` (or your preferred name)
3. Description: "AI-powered CLI tool for code review that suggests improvements and catches bugs"
4. Set to Public (or Private if you prefer)
5. **DO NOT** initialize with README, .gitignore, or license (we already have these)
6. Click "Create repository"

## Option 2: Push to Existing Repository

If you already have a repository, use its URL.

## Push Commands

After creating the repository, run these commands:

```bash
cd /Users/yoshikondo/ai-code-review

# Add remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/ai-code-review.git

# Or if using SSH:
# git remote add origin git@github.com:YOUR_USERNAME/ai-code-review.git

# Push to GitHub
git push -u origin main
```
