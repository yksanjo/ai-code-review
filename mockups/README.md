# UX/UI Mockups

This directory contains HTML mockups for all 7 developer tools. These can be opened in a browser and screenshotted for use in documentation.

## Files

1. **index.html** - Overview page with links to all mockups
2. **1-ai-code-review.html** - AI Code Review Assistant CLI interface
3. **2-api-docs.html** - API Documentation Generator website
4. **3-git-commit.html** - Smart Git Commit Generator CLI
5. **4-dev-setup.html** - Dev Environment Setup Tool wizard
6. **5-codebase-qa.html** - Codebase Question Answerer interface
7. **6-error-translator.html** - Error Message Translator browser extension
8. **7-test-generator.html** - Test Case Generator CLI

## How to Use

1. Open `index.html` in your browser to see all mockups
2. Click on any mockup to view it
3. Take screenshots using:
   - Mac: `Cmd+Shift+4` (select area) or `Cmd+Shift+3` (full screen)
   - Windows: `Win+Shift+S` (Snipping Tool)
   - Browser: Right-click → Inspect → Device Toolbar → Screenshot
4. Save screenshots to `screenshots/` directory
5. Update PRODUCTS.md to include image references

## Screenshot Tips

- Use browser zoom (Cmd/Ctrl + Plus) to make mockups larger
- Use browser DevTools to adjust viewport size
- Consider taking screenshots at different sizes (desktop, tablet, mobile)
- Use consistent naming: `product-name-view.png`

## Integration with PRODUCTS.md

After taking screenshots, update PRODUCTS.md to include:

```markdown
### UX/UI Design

![AI Code Review CLI](mockups/screenshots/1-ai-code-review.png)
```
